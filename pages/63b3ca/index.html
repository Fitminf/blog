选择排序

# 选择排序

选择排序就是对一个数组或者集合进行特定规则的排序，例如一个数组的升序排序，原理是从第一个值 a 开始和后面每一个值 n 进行比较，假设 a > n 则把 a 的位置不变，假设 a < n 则 a 和 n 的值互换位置，以此类推。

/**
 * 选择排序基础实现
 * <p>
 * 每一轮找出最小值排序
 * <p>
 * O(n²)
 */
public static class SelectionSortBaseInnerClass {
    public static void main(String[] args) {
        int[] arr = AlgorithmUtil.intArrays(200000, false);                // O(n)
        long start = System.currentTimeMillis();
        for (int j = 0; j < arr.length - 1; j++) {
            int minIndex = j;
            for (int i = j + 1; i < arr.length; i++) {          // O(n²)
                // 每一轮找出最小值排序
                if (arr[minIndex] > arr[i]) {
                    minIndex = i;
                }
            }
            int temp = arr[j];
            arr[j] = arr[minIndex];
            arr[minIndex] = temp;
        }
        long end = System.currentTimeMillis();
        System.out.println("时间消耗：" + (end - start) + " ms");
        // System.out.println(Arrays.toString(arr));
        // 200000 ==> 12.5 s
    }
}

在这里 AlgorithmUtil.intArrays 是我自己写的一个根据参数传入的长度 length 创建一个长度为 length 的数组返回。false 是表示创建这个数组后是否打印这个数组，由于测试的长度太大就设置为 false。

在这个例子中可以优化一下，让每次循环不仅找到最小值也找到最大值，于是有了下面的写法：

/**
 * 选择排序优化
 * <p>
 * 每一轮找出最小值和最大值排序排序
 * O(n²)
 */
public static class SelectionSortOptimize01InnerClass {
    public static void main(String[] args) {
        int[] arr = AlgorithmUtil.intArrays(200000, false);                // O(n)
        long start = System.currentTimeMillis();
        for (int j = 0; j < (arr.length / 2) + 1; j++) {
            int minIndex = j;
            int maxIndex = arr.length - 1;
            for (int i = j + 1; i < arr.length; i++) {          // O(n²)
                // 每一轮找出最小值和最大值排序排序
                if (arr[minIndex] > arr[i]) {
                    minIndex = i;
                }
                if (arr[maxIndex] < arr[i]) {
                    maxIndex = i;
                }
            }
            int temp1 = arr[j];
            arr[j] = arr[minIndex];
            arr[minIndex] = temp1;
            int temp2 = arr[arr.length - 1];
            arr[arr.length - 1] = arr[maxIndex];
            arr[maxIndex] = temp2;
        }
        long end = System.currentTimeMillis();
        System.out.println("时间消耗：" + (end - start) + " ms");
        // System.out.println(Arrays.toString(arr));
        // 200000 ==> 8.5 s
    }
}
