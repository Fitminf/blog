发布中央仓库

# 发布中央仓库

2024 改版

# 账号准备

# 注册

在 https://central.sonatype.com/ 网站上注册账号

# 新建命名空间

登录后在顶部找到 Publish（发布） 按钮点击

找到 Namespaces（名称空间） 按钮点击

找到 Add Namespaces 按钮点击

添加命名空间

  • 使用 gitee 作为命名空间
  • 比如我的 gitee 主页链接是：https://gitee.com/fitminf
  • 那么我的命名空间是 io.gitee.fitminf
    • 其他的仓库可以按照下面的要求来创建：
      • GitHub io.github.myusername
      • GitLab io.gitlab.myusername
      • ...

验证命名空间

  • 点击 verify namespace 的名称
  • 按照要求创建一个仓库以证明该命名空间属于你
  • 之后点击 confirm 即可完成

创建 push 账号密码

  • 点击右上角 view account
  • 点击 Generate User Token
  • 点击 ok
  • 会生成一个账号密码，在一分钟内记录下来（一分钟以后弹窗会消失需要重新生成）

账号准备自此完毕

# Gradle

使用 vanniktech 的插件

  • 官网：https://github.com/vanniktech/gradle-maven-publish-plugin
  • 文档：https://vanniktech.github.io/gradle-maven-publish-plugin/central/

最终

build.gradle.kts 文件如下：

import com.vanniktech.maven.publish.SonatypeHost

description = "tailor-core script configuration"

version = "1.0.7"


plugins {
    id("com.vanniktech.maven.publish") version "0.28.0"
}


// 定义源代码和JavaDoc的Jar任务
mavenPublishing {
    // or when publishing to https://central.sonatype.com/
    publishToMavenCentral(SonatypeHost.CENTRAL_PORTAL)
    signAllPublications()

    coordinates("io.gitee.fitminf", "tailor-core", "1.0.7")

    pom {
        name.set("tailor-core")
        description.set("test publish")
        inceptionYear.set("2024")
        url.set("https://gitee.com/fitminf/caramel-personal-tailor")
        licenses {
            license {
                name.set("The Apache License, Version 2.0")
                url.set("http://www.apache.org/licenses/LICENSE-2.0.txt")
                distribution.set("http://www.apache.org/licenses/LICENSE-2.0.txt")
            }
        }
        developers {
            developer {
                id.set("caramel-personal-tailor")
                name.set("tailor-core")
                email.set("fitminf@163.com")
            }
        }
        scm {
            url.set("https://gitee.com/fitminf/caramel-personal-tailor")
            connection.set("scm:git:git://gitee.com/fitminf/caramel-personal-tailor.git")
            developerConnection.set("scm:git:ssh://gitee.com/fitminf/caramel-personal-tailor.git")
        }
    }
}
